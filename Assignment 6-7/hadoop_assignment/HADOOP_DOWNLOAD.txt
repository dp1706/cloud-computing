------------------------------------------------------------------------------------------------------------------------------------------------------                                                               DOWNLOAD HADOOP
------------------------------------------------------------------------------------------------------------------------------------------------------



UBUNTU VERSION :  18.04
JAVA PRE-INSTALLED VERSION : 11.0.6( TO install -- sudo apt-get install default-jdk )

Commands:

1. sudo apt-get install update
2. sudo apt-get update
3. sudo wget http://mirrors.estointernet.in/apache/hadoop/common/hadoop-2.10.0/hadoop-2.10.0.tar.gz 
4. tar -xzvf hadoop-2.10.0.tar.gz
5. sudo mv hadoop-2.10.0 /usr/local/hadoop
6. readlink -f /usr/bin/java | sed "s:bin/java::"
7. export JAVA_HOME=$(readlink -f /usr/bin/java | sed "s:bin/java::")
8. export PATH=$PATH:$JAVA_HOME/bin
9. export PATH=$PATH:/usr/local/hadoop/bin

---To find version and check hadoop is installed properly---

10. hadoop version

----command for running wordcount mapper and reducer file which is in the hadoop_assignment directory

11. hadoop jar /usr/local/hadoop/share/hadoop/tools/lib/hadoop-streaming-*.jar -input /home/dwarka/hadoop_assignment/input.txt -output /home/dwarka/hadoop_assignment/output -mapper /home/dwarka/hadoop_assignment/mapper.py -reducer /home/dwarka/hadoop_assignment/reducer.py


-----------------done    -----------------
